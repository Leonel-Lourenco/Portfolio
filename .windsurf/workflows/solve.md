## Objective
Systemically solve problems by identifying root causes and implementing elegant solutions.

## Requirements
For every problem encountered:
1. Generate 5 probable causes ranging from obvious to subtle
   - Cause 1: Painfully obvious (typos, missing imports, etc.)
   - Cause 2: Simple (configuration, environment setup)
   - Cause 3: Moderate (logic errors, state management)
   - Cause 4: Complex (race conditions, architectural issues)
   - Cause 5: Subtle (edge cases, implicit dependencies)
2. Identify the actual root cause through investigation
3. Implement an elegant, minimal solution that addresses the root cause
4. Verify the fix resolves the problem without introducing new issues

## Deliverable
A working solution where the root cause is fixed, not just the symptoms.
